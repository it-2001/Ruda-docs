Installation

Note Ruda is still in development, so there is no official release yet. However, you can still install it from source.

Compiler is still in development, and it is not yet possible to compile Ruda programs at the moment.

Prerequisites

To build Ruda, you need to have the following installed:

  • Rust

Cloning the repositories

  • compiler
  • virtual machine
  • std.io
  • std.string

Building

Go to the compiler directory and modify the Cargo.toml file to point to the correct paths of the stdlib repositories and VM. Then run the following command:

cargo build

This will build the compiler and place the executable in the target/debug directory.

For VM and stdlib, run the following command in their respective directories:

cargo build

This will build the VM and stdlib and place the executables in the target/debug directory.

Now you should be able to run the compiler.
