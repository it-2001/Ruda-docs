Number guessing game

Well technicaly this is your second program, but it's the first one that does something interesting.

In this project you will create a number guessing game. The computer will a predefined number between 1 and 100 and you will try to guess it. The computer will tell you if your guess is too high or too low.

Requirements

  • The user should be able to guess the number.
  • The computer should tell the user if their guess is too high or too low.
  • The user should be able to guess again if they guess incorrectly.
  • The user should be able to quit the game at any time.
  • The user should be told how many guesses it took them to guess the number when they win.

Example

Welcome to the number guessing game!
I am thinking of a number between 1 and 100.
Guess the number: 50
Too high!
Guess the number: 25
Too low!
Guess the number: 37
Too high!
Guess the number: 31
Too high!
Guess the number: 28
Too low!
Guess the number: 29
You win! It took you 6 guesses.

Tips

  • You can use the io.input function to read a line of input from the user.
  • You can use the io.println function to print a line of output to the user.
  • The generated number can be constant.

Strategy

This is a good time to start thinking about how you will structure your code.

In my head I'm focusing on the following parts:

  • Ability to ask the user for input until they guess the correct number.
  • Changing the game answer based on the user's input.
  • Keeping track of how many guesses the user has made.
  • Telling the user how many guesses they made when they win.

Solution

I hope you are done with the project before you look at this solution. If you are stuck, try to get unstuck before looking at the solution.

import "std.io"

const answer: int = 29

fun main() {
    let guesses = 0

    io.println("Welcome to the number guessing game!")
    io.println("I am thinking of a number between 1 and 100.")

    loop {
        io.print("Guess the number: ")
        let guess = io.input() as int

        guesses += 1

        if guess < answer {
            io.println("Too low!")
        } else if guess > answer {
            io.println("Too high!")
        } else {
            io.println("You win! It took you " + guesses as string + " guesses.")
            break
        }
    }
}

Let's go over the solution.

First we create a constant called answer and set it to 29. This is the number that the user will try to guess.

const answer: int = 29

Next we create a variable called guesses and set it to 0. This variable will keep track of how many guesses the user has made.

let guesses = 0

Next we print a welcome message to the user.

This message is outside of the loop because we only want to print it once.

io.println("Welcome to the number guessing game!")
io.println("I am thinking of a number between 1 and 100.")

Next we create a loop. This loop will keep asking the user for input until they guess the correct number.

loop {
    ...
}

Inside the loop we ask the user for input and store it in a variable called guess.

Here I cheat a little bit. The io.input function returns a string, but we want to store it in an integer variable. To do this we use the as keyword to cast the string to an integer.

io.print("Guess the number: ")
let guess = io.input() as int

Next we increment the guesses variable by 1 to keep track of how many guesses the user has made.

guesses += 1

Next we check if the user's guess is too low. If it is we print a message to the user.

if guess < answer {
    io.println("Too low!")
}

Next we check if the user's guess is too high. If it is we print a message to the user.

if guess > answer {
    io.println("Too high!")
}

Next we check if the user's guess is correct. If it is we print a message to the user and break out of the loop.

if guess == answer {
    io.println("You win! It took you " + guesses as string + " guesses.")
    break
}

This is the end of the loop. The loop will repeat until the user guesses the correct number.
