Pointers

What is a pointer?

A pointer is a variable that stores the address of another variable. In other words, a pointer points to another variable.

How to declare a pointer?

To declare a pointer, we use the & operator.

Example:

fun main() {
    let a = 5
    let b = &a
    io.println(b)
}

Output:

0x0000000000000001

In this example, we declare a variable a and assign it the value 5. We then declare a variable b and assign it the address of a. We then print the value of b which is the address of a.

Dereferencing a pointer

To dereference a pointer, we use the * operator.

Example:

fun main() {
    let a = 5
    let b = &a
    io.println(*b)
}

Output:

5

In this example, we declare a variable a and assign it the value 5. We then declare a variable b and assign it the address of a. We then print the value of b which is the address of a. We then dereference b and print the value of a.

Pointers to pointers

Pointers can point to other pointers.

Example:

fun main() {
    let a = 5
    let b = &a
    let c = &b
    io.println(*c)
}

Output:

0x0000000000000001

In this example, we declare a variable a and assign it the value 5. We then declare a variable b and assign it the address of a. We then declare a variable c and assign it the address of b. We then print the value of c which is the address of b. We then dereference c and print the value of b.

Pointers to structs

Pointers can point to structs.

Example:

struct Vector {
    x: float,
    y: float,
}

impl Vector {
    constructor(x: float, y: float) {
        this.x = x
        this.y = y
    }
}

fun main() {
    let a = Vector(1, 2)
    let b = &a
    io.println(*b.x)
    io.println(*b.y)
}

Output:

1
2

Type of a pointer

The type of a pointer is &T where T is the type of the variable it points to.

Example:

fun main() {
    let a = 5
    let b: &int = &a
}

Here b is a pointer to an int. Pointer to pointer would be &&T and so on.
