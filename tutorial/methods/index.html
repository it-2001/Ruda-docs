Methods

Methods are functions that are associated with a type. To implement a method, use the impl keyword followed by the type name, then the method name. The method body is enclosed in curly braces.

struct Person {
    name: string,
    age: int
}

impl Person {
    constructor(name: string, age: int) {
        self.name = name
        self.age = age
    }

    fun greet(self) {
        io.println("Hello, my name is " + self.name)
    }
}

We have already seen constructors. Constructors are special methods that are called when a struct is created. Constructors are used to initialize the struct fields.

Methods are called using the dot operator.

let person = Person("John", 20)

person.greet()

Output:

Hello, my name is John

Self

Self is a special variable that refers to the struct instance. We use self to access the struct fields.

Static methods

If you do not have self as a parameter, compiler will treat the method as a static method. Static methods are methods that are not associated with a struct instance. Static methods are called using the type name.

struct Person {
    name: string,
    age: int
}

impl Person {
    constructor(name: string, age: int) {
        self.name = name
        self.age = age
    }

    fun average_age(): float {
        // some magic to calculate average age of all people
        // ...

        return average_age
    }
}

let person1 = Person("John", 20)

let person2 = Person("Jane", 30)

let average_age = Person.average_age()

Here we have a static method named average_age. This method calculates the average age of all people. We can call this method using the type name.

We want it to be a static method because it is not associated with a struct instance. It is associated with the struct itself.

Static methods vs functions

Static methods are similar to functions. The only difference is that static methods are associated with a type.
