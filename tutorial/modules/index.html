Modules

This will be a short tutorial on how modularity works in Ruda.

What is a module?

A module is a file that contains code. Modules can be imported into other modules. This allows you to use the code from one module in another module.

Creating a module

To create a module, create a file with the .ruda extension. This is the extension used for Ruda modules.

Example:

// This is a module named "hello_world.ruda"

const MESSAGE: string = "Hello world"

Importing a module

To import a module, use the import keyword followed by the name of the module.

Example:

import "hello_world"

fun main() {
    io.println(hello_world.MESSAGE)
}

Output:

Hello world
