Control Flow

Control flow lets you control the order in which your code is executed. It allows you to add logic to your programs.

Conditional Statements

Conditional statements are used to execute certain code only if a certain condition is met.

if x {
    // code to execute if condition is true
}

Here x is the condition. If x is true, the code inside the curly braces is executed. If x is false, the code is skipped.

Example:

let x = 5

if x == 5 {
    io.println("x is 5")
}

Output:

x is 5

Here we check if the value of x is 5. Since it is, the code inside the curly braces is executed.

Else

You can use the else keyword to execute code if the condition is false.

if x {
    // code to execute if condition is true
} else {
    // code to execute if condition is false
}

Example:

let x = 5

if x == 5 {
    io.println("x is 5")
} else {
    io.println("x is not 5")
}

Output:

x is 5

Here we check if the value of x is 5. Since it is, the code inside the first curly braces is executed. The code inside the second curly braces is skipped.

If we change the value of x to 6, the code inside the first curly braces is skipped and the code inside the second curly braces is executed.

let x = 6

if x == 5 {
    io.println("x is 5")
} else {
    io.println("x is not 5")
}

Output:

x is not 5

Else If

You can use the else if keyword to check multiple conditions.

if x {
    // code to execute if condition is true
} else if y {
    // code to execute if condition is true
} else {
    // code to execute if all conditions are false
}

Example:

let x = 6

if x == 5 {
    io.println("x is 5")
} else if x == 6 {
    io.println("x is 6")
} else {
    io.println("x is not 5 or 6")
}

Output:

x is 6

Here we check if the value of x is 5. Since it is not, we check if the value of x is 6. Since it is, the code inside the second curly braces is executed. The code inside the first and third curly braces is skipped.

If we change the value of x to 7, the code inside the first and second curly braces is skipped and the code inside the third curly braces is executed.

Switch Statements

Switch statements are used to execute certain code based on the value of a variable.

switch x {
    y {
        // code to execute if x == y
    }
    z {
        // code to execute if x == z
    }
    _ {
        // code to execute if neither case is true
    }
}

Here x is the variable to check. If x is equal to y, the code inside the first curly braces is executed. If x is equal to z, the code inside the second curly braces is executed. If x is not equal to either y or z, the code inside the third curly braces is executed.

Example:

let x = 6

switch x {
    5 {
        io.println("x is 5")
    }
    6 {
        io.println("x is 6")
    }
    _ {
        io.println("x is not 5 or 6")
    }
}

Output:

x is 6

Loops

Loops are used to execute code multiple times.

Loop

The loop loop executes a block of code forever.

loop {
    // code to execute forever
}

Example:

loop {
    io.println("Hello world")
}

Output:

Hello world
Hello world
Hello world
Hello world
(etc.)

While

The while loop executes a block of code while a condition is true.

while x {
    // code to execute while condition is true
}

Example:

let x = 0

while x < 5 {
    io.println(x)
    x = x + 1
}

Output:

0
1
2
3
4

For

The for loop unitizes Ruda's iterators to execute a block of code for each item in an iterator.

for x in y {
    // code to execute for each item in y
}

Example:

for character in "Hello" {
    io.println(x)
}

Output:

H
e
l
l
o

Break

The break keyword is used to exit a loop.

loop {
    // code to execute forever
    break
}

Example:

let x = 0

while true {
    io.println(x)
    x = x + 1

    // exit loop when x is 5
    if x == 5 {
        break
    }
}

Output:

0
1
2
3
4

Here we use the break keyword to exit the loop when x is 5.

Continue

The continue keyword is used to skip the rest of the code in a loop and continue to the next iteration.

loop {
    // code to execute forever
    continue
}
