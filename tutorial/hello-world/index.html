Hello world

Where else to start than with the classic "Hello world" example.

import "std.io"

fun main() {
    io.println("Hello world")
}

Let's break this down line by line.

import "std.io"

Ruda on its own is a very small language. It only has a few built-in functions and types. To get access to more functionality, you need to import modules. In this case, we're importing the standard library's IO module. This module contains functions for reading and writing to the console.

fun main() {

This is the main function. Every Ruda program needs to have a main function. It's the entry point of the program.

io.println("Hello world")

This line calls the println function from the IO module. It prints the string "Hello world" to the console.

Running the program

Ruda programs are compiled to Ruda bytecode. This bytecode is then run by the Ruda virtual machine. This means that you need to have the Ruda virtual machine installed to run Ruda programs.

To compile and run the program, run the following command in the terminal:

ruda run hello_world.ruda

This will compile the program and run it. You should see the following output:

Hello world

Congratulations! You've just run your first Ruda program!
