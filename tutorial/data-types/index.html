Data types

Primitive types

Type Description example
bool A boolean value. true or false
int A signed integer. 5, -5
float A floating point number. 5.0, -5.0
char A single unicode character. 'a', '\n'
string A string of unicode characters. "Hello, world!"
usize An unsigned integer the size of a pointer. 5
pointer A pointer to a value. &5
function A function. Sometimes you want to pass a function as an argument to another function. let f = fun(x: int): int { return x + 1 }

Compound types

Type Description example
array A fixed-size array of values. [1, 2, 3, 4, 5] or [1; 5]
struct A struct of values. struct Point { x: int, y: int }
enum An enum of values. enum Compass { North, South, East, West }

Type aliases

Type aliases are used to give a type a different name. They are often used to abstract away implementation details.

type int = i32

Type casting

You can cast a value to a different type using the as keyword.

let x: int = 5
let y: float = x as float

Type inference

Ruda has type inference. This means that the compiler can infer the type of a value based on its usage.

let x = add(5, 5)

In this example, the compiler can infer that x is an int because add returns an int.

Optional types

Optional types are used to represent a value that may or may not exist. They are useful for handling errors.

let x: int? = 5

if x? {
    io.println(x)
} else {
    io.println("x is not defined")
}
