Operators

Arithmetic Operators

Sometimes you need to perform arithmetic operations on variables. For example, you may need to increment or decrement a variable's value. The following arithmetic operators are supported by the language:

Operator Description Example
+ Addition x + y
- Subtraction x - y
* Multiplication x * y
/ Division x / y
% Modulo x % y

Example:

let x = 5
let y = 2
let z = x + y

Statements can also be grouped using parentheses. For example:

let x = 5
let y = 2
let z = (x + y) * 2

Here the expression (x + y) is evaluated first, then the result is multiplied by 2.

Assignment Operators

Assignment operators are used to assign values to variables. The following assignment operators are supported by the language:

Operator Description Example
= Assign x = y
+= Add and assign x += y
-= Subtract and assign x -= y
*= Multiply and assign x *= y
/= Divide and assign x /= y

Example:

let x = 5
x += 2

We use the += operator to add 2 to the value of x. This is equivalent to x = x + 2. If we print the value of x now, we get 7.

Comparison Operators

Comparison operators are used to compare values. The following comparison operators are supported by the language:

Operator Description Example
== Equal x == y
!= Not equal x != y
< Less than x < y
> Greater than x > y
<= Less than or equal x <= y
>= Greater than or equal x >= y

Example:

let x = 5
let y = 2
let z = x > y

Here we compare the values of x and y using the > operator. Since x is greater than y, the result is true.

Logical Operators

Logical operators are used to combine boolean expressions. The following logical operators are supported by the language:

Operator Description Example
&& Logical AND x && y
|| Logical OR x || y
! Logical NOT !x

Error Handling Operators

Error handling operators are used to handle errors. The following error handling operators are supported by the language:

Operator Description Example
? exists x? returns true if x is not null
! error see the error handling section for more information

Example:

let x = true
let y = false
let z = x && y

Here we combine the boolean values of x and y using the && operator. Since x is true and y is false, the result is false.
