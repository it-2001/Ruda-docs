Constants

Constants are values that cannot be changed. They are declared using the const keyword.

Unlike variables, constants must be assigned a value when they are declared. And they can be Accessed anywhere in the file or another file that imports the file they are declared in.

const PI: float = 3.14

This declares a constant named PI and assigns it the value 3.14.

Naming conventions

Constants are written in UPPER_SNAKE_CASE.

Example:

const WINDOW_WIDTH: int = 800
const WINDOW_HEIGHT: int = 600

Why use constants?

Constants are useful for values that are used multiple times in a program. They make it easier to change the value of something in one place instead of having to change it in multiple places.

By using constants, you can also give names to values that would otherwise be hard to understand.

When you see WINDOW_WIDTH in a program, you know that it is the width of the window. If you see 800 instead, you have no idea what it is.

Another advantage of constants is that modules can export them. This allows other modules to use them.
