Enums

Enums are a way to define a type that can have a finite number of values. For example, a boolean can only be true or false, so it can be represented as an enum:

enum bool {
    true,
    false,
}

Enums can also have associated numbers:

enum Compass {
    North = 1,
    South = 10,
    East = 100,
    West = 1000,
}

Variant matching

Enum variants are treated as values, so you can match on them:

enum Compass {
    North,
    South,
    East,
    West,
}

let direction = Compass.North

switch direction {
    Compass.North {
        io.println("North")
    },
    Compass.South {
        io.println("South")
    },
    Compass.East {
        io.println("East")
    },
    Compass.West {
        io.println("West")
    }
}

Output:

North

Variant values

Since enum variants can be associated with numbers, you can get the value of a variant:

enum Compass {
    North = 1,
    South = 10,
    East = 100,
    West = 1000,
}

let direction = Compass.North

let value = direction as int

io.println(value)

Output:

1

And by casting them to string, you can get their name:

enum Compass {
    North = 1,
    South = 10,
    East = 100,
    West = 1000,
}

let direction = Compass.North

let name = direction as string

io.println(name)

Output:

North

Constructors

Enums also allow you to define constructors, but they are not as useful as struct constructors:

enum Compass {
    North = 1,
    South = 10,
    East = 100,
    West = 1000,
}

impl Compass {
    constructor() {
        self = Compass.North
    }
}

let direction = Compass()

io.println(direction as string)

Output:

North
