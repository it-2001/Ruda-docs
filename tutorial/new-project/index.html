New project

Create a new project

To create a new project, you can use the init command:

ruda init

This will create a new project in the current directory.

Project structure

A Ruda project consists of the following files:

  • src/main.rd - The main file of the project. This is where the main function is defined.
  • Ruda.toml - A file that contains information about the project.
  • .gitignore - A file that tells Git which files to ignore.

Building the project

To build the project, you can use the build command:

ruda build

This will compile the project and create an executable file in the target directory.

Running the project

To run the project, you can use the run command:

ruda run

This will compile the project and run the executable file in the target directory.

In this case the output will be:

Hello world

This is all you need to know to get started with Ruda.
