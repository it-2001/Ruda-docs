Functions

Functions are useful for writing code that you want to reuse. They can be defined anywhere in the file, and can be called anywhere in the file. You already know how to use functions, because main is a function.

Defining a function

Functions are defined using the fun keyword. The function name comes after the fun keyword. The function body is enclosed in curly braces.

fun greet() {
    io.println("Hello world")
}

Calling a function

Functions are called by writing the function name followed by parentheses.

greet()

Output:

Hello world

Parameters

Functions can take parameters. Parameters are variables that are passed to the function when it is called. Parameters are enclosed in parentheses after the function name and have specified types.

Multiple parameters are separated by commas.

fun greet(name: string) {
    io.println(name)
}

This function takes a parameter called name of type string. The function body prints the name to the console.

Call the function like this:

greet("John")

Output:

John

Here we are passing the string "John" to the function where it can be used as the name parameter.

Return values

Functions can return values. The return type is specified after the parameter list. The return type can be any type.

fun add(a: int, b: int): int {
    return a + b
}

This function takes two parameters of type int and returns their sum.

Call the function like this:

let result = add(1, 2)

Here we are passing the integers 1 and 2 to the function where they can be used as the a and b parameters. The function returns the sum of the two numbers, which is then stored in the result variable.
