Keywords

Ruda has a small number of keywords. They are listed below.

Variable declaration

Keyword Description example reference
let Declares a variable. let x = 5 or let x: int = 5 Variables
const Declares a constant. const x: int = 5 Constants
type Declares a type alias. type MyInt = int Type aliases

Control flow

Keyword Description example reference
if Executes a block of code if a condition is true. if x == 5 { ... } If
else Executes a block of code if a condition is false. if x == 5 { ... } else { ... } Else
else if Executes a block of code if a condition is false and another condition is true. if x == 5 { ... } else if x == 6 { ... } Else if
switch Executes a block of code depending on the value of a variable. switch x { 1 { ... }, _ { ... } } Switch

Loops

Keyword Description example reference
while Executes a block of code while a condition is true. while x < 5 { ... } While
for Executes a block of code using an iterator. for x in [0, 1, 2, 3, 4, 5] { ... } For
loop Executes a block of code indefinitely. loop { ... } Loop
break Breaks out of a loop. break Break
continue Skips the rest of the current iteration of a loop. continue Continue

Functions

Keyword Description example reference
fun Declares a function. fun add(x: int, y: int) int { ... } Functions
return Returns a value from a function. return 5 Return
overload Declares an operator overload. overload + (x: int, y: int) int { ... } Operator overloading

Comments

Keyword Description example reference
// Single-line comment. // This is a comment Comments
/* */ Multi-line comment. /* This is a comment */ Comments

Non primitive types

Keyword Description example reference
struct Declares a struct. struct Point { x: int, y: int } Structs
enum Declares an enum. enum Color { Red, Green, Blue } Enums
impl Declares an implementation block. impl Point { ... } Implementations
trait Declares a trait. trait Drawable { ... } Traits
import Imports a module. import "std.io" Modules

Memory management

Keyword Description example reference
new Allocates memory on the heap. new Point { x: 5, y: 5 } Memory management
as Casts a value to a different type. x as float Casting
? Optional operator. Checks if value is not null. if maybeValue? { ... } Optional operator

Error handling

Keyword Description example reference
try Executes a block of code that may fail. try { ... } Try
catch Handles an error. catch { ... } Catch
yeet Throws an error. yeet Error("Something went wrong") Yeet
error Declares an error type. error Error (msg: string) { message: msg, code: 1 } Error
! Bang operator. unsafeFunction()! Bang
